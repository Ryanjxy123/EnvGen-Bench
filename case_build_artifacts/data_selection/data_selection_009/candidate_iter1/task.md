Create a Library Week spotlight card that recommends the highest-priority resources for patrons.

Use the workbook sheets and rule file to shortlist resources for the campaign. The card should show the selected resources in priority order, using reader-friendly category and branch labels from the mapping sheets rather than raw codes. Do not feature draft, archived, cancelled, wrong-window, rights-failed, low-stock, high-handling, group-cap-blocked, or lower-ranked overflow resources.
