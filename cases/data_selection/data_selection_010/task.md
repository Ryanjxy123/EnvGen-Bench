Create a winter workshop recommendation flyer for community members.

Use the SQLite database and recommendation rules to choose the workshops to promote. The database uses category and district codes, so join to the support tables before deciding eligibility and before writing display labels. Feature selected workshops in recommendation order with category, district, seat capacity, and time band. Do not promote archived, draft, cancelled, wrong-term, readiness-failed, threshold-failed, category-cap-blocked, or lower-ranked workshops.
